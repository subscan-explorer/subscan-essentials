## Description

### Global Header 

| Name         | Type             | Require |
|--------------|------------------|---------|
| Content-Type | application/json | yes     |


#### List row&page

| Name | Type | Desc                                                                |
|------|------|---------------------------------------------------------------------|
| row  | int  | min=1,max=100 defines how many results should be presented per page |
| page | int  | min=0 defines which page should be shown                            |
